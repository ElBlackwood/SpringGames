CalTri Fitness Android Application

Usage:

- Record Triathlon training workouts - cycle, swim or run
- Add name, date, distance, time, comments and workout intesity to each entry with record screen
- All entries are stored on the Training Log screen for future viewing (entries can be deleted and sorted here)
- Check monthly progress with a visual representation on the Graph Screen (more options/visualisations to be implemented shortly)
- Sync - Save a copy of your training log via email in .csv format for future viewing (UI improvements/confirmations coming shortly)
